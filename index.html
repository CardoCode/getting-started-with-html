The World Wide Web

The web was invented in the 1990's and is made up of pages written in HTML or Hypertext Markup Language. The components to accessing the web are: you, a computer and browser, internet access using HTTP, and servers.

HTML

HTML can be:
Text - what you see
Markup Elements - what it looks like - (uses Tags like b for bold) Elements can be inline or block.
References to other docs like images and videos
Links to other pages
